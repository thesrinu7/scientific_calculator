## Overview

The Scientific Calculator is a Java-based desktop application developed using Java Swing. It provides both basic arithmetic operations and scientific functions through an intuitive graphical user interface (GUI). This project demonstrates the application of Core Java concepts, event-driven programming, and GUI development.

## Features

### Basic Operations
- Addition
- Subtraction
- Multiplication
- Division

### Scientific Functions
- Square Root
- Square (x²)
- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Logarithm (log)
- Percentage Calculation

### User Interface
- Interactive GUI built with Java Swing
- Easy-to-use button layout
- Real-time calculation display
- Error handling for invalid inputs

## Technologies Used

- Java
- Java Swing
- AWT Event Handling
- Eclipse IDE / VS Code

## Installation and Execution

### Prerequisites

- Java JDK 8 or later
- Eclipse IDE or VS Code

### Steps to Run

1. Clone the repository:

```bash
git clone https://github.com/thesrinu7/scientific_calculator.git
```

2. Navigate to the project directory.

3. Compile the source file:

```bash
javac ScientificCalculator.java
```

4. Execute the application:

```bash
java ScientificCalculator
```
